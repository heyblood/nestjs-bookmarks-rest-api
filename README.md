# NestJS Bookmarks REST API

Build a bookmarks REST API from scratch using Nest.js, Docker, PostgreSQL, Passport.js, Prisma, PactumJS and Dotenv.

## Installation

```bash
# install app dependencies
$ yarn install

# setup postgres database with prisma
$ yarn run db:dev:restart

```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
